Text edictor 是 Erick lang（梁业淼，云南大学软件学院数字媒体技术专业） 开发的轻量化文本编辑软件

该软件使用 python 语言和 python 库 tkinter 编写，在tkinter的Text组件上实现系列功能，包括选中字体样式设置，字体前景颜色、背景颜色等处理，并为文本编辑功能添加了行号支持，行号支持功能可以智能追踪用户操作，比如撤销、粘贴、选中删除等系列操作，根据这些操作同步更新行号。除了文本编辑界面的特性，本人还为text edictor添加了优美的软件launch界面，通过该登录界面可以进行打开文件、保存文件以及更改软件界面壁纸等操作来自定义自己风格的text edictor

功能介绍

launch界面功能

新建文件

​ 点击菜单栏 file-> new file来打开新的edict界面来进行文本编辑。

打开文件

​ 点击菜单栏file->open file来选取文件，然后使用text edictor打开文件，对文件进行读写操作。

保存文件

​ edict界面，点击工具栏内return按钮，返回launch界面，再点击菜单栏file->save file来打开text edictor的文件保存路径设置界面，该界面内，用户可以保存文件到任何自定义路径下。

文件另存为

​ edict界面，点击工具栏内return按钮，返回launch界面，再点击菜单栏file->save file as来打开文件资源管理器来选择文件保存路径。

设置默认壁纸

​ 点击菜单栏的 config->set default wallpaper来设置默认壁纸，此后，每次从新打开text edictor软件看到的壁纸都将是用户设置的默认壁纸。

添加壁纸

​ 点击菜单栏的config->add my wallpaper，打开文件资源管理器，将用户选择的图片添加到软件中，用户可通过图片索引（新添加的图片索引再之前软件内已存在的壁纸的索引之后）来设置text edictor的launch界面壁纸为新添加的图片。

help文档

​ 点击菜单栏的help菜单项，使用浏览器解析并打开用户帮助文档
